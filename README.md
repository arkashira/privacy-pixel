# Image Editor
A simple image editor with redaction capabilities.

## Features
* Apply redaction to images using a blur or pixelate brush
* Toggle redacted areas on and off
* Get the redacted image

## Usage
1. Create an instance of the `ImageEditor` class with the image data.
2. Apply redaction to the image using the `apply_redaction` method.
3. Toggle redacted areas on and off using the `toggle_redaction` method.
4. Get the redacted image using the `get_redacted_image` method.
