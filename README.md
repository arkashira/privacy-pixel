# 🚀 privacy-pixel

**Power developers with simple image redaction.** A simple image editor for applying redaction to images using blur or pixelate brushes.

## Why privacy-pixel?

* **Easy to use**: A simple image editor that provides redaction capabilities.
* **Flexible**: Toggle redacted areas on and off.
* **Secure**: Get the redacted image.
* **Built for developers**: A tool for developers or users needing a simple image redaction tool.
* **Fast**: Built using Python and Poetry for a fast development cycle.
* **Maintainable**: A well-structured codebase with clear documentation.

## Feature Overview

| Feature | Description |
| --- | --- |
| Redaction | Apply redaction to images using blur or pixelate brushes. |
| Toggle | Toggle redacted areas on and off. |
| Get Redacted Image | Get the redacted image. |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/axentx/privacy-pixel.git

# Install dependencies
poetry install

# Run the application
poetry run python src/main.py
```

## Deploy

```bash
# Build the application
poetry build

# Deploy to your desired platform
# (e.g. AWS, GCP, etc.)
```
