# Privacy Pixel Project
This project aims to provide a simple implementation of a privacy pixel with accessibility features.

## Features
* Keyboard navigation
* ARIA labels for interactive elements
* WCAG 2.1 AA contrast ratio compliance

## Requirements
* Python 3.9 or later

## Installation
1. Clone the repository
2. Install dependencies using `poetry install`

## Running Tests
1. Install dependencies using `poetry install`
2. Run tests using `poetry run pytest`
